# 🎁 Caja Misteriosa - Multijugador

Juego de apuestas online donde los jugadores eligen cajas misteriosas para ganar el pozo acumulado.

## Estructura

- `/public` - Vista de jugadores (sin controles de admin)
- `/admin` - Panel de administración
- `/api` - Backend Node.js/Express

## Instalación Local

```bash
npm install
npm start
